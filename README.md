# Історія погоди по містах 🌤️

Проєкт дозволяє переглядати історичні дані про погоду у різних містах. Користувач може обрати дату та місто і отримати детальну інформацію про погодні умови того дня.

## 🔹 Можливості

- Пошук погоди за конкретним містом(можна писати різними мовами)
- Вибір дати для отримання історичних даних(поки що, немає даних за останній місяць, тож раджу вводити дату за 2 місяці раніше ніж поточна, цей функціонал буде допрацьовуватись)
- Відображення інформації про температуру та інші параметри
- Адаптивний інтерфейс для будь-якого пристрою

## 💻 Технології

- HTML, CSS, JavaScript
- API для отримання даних про погоду 
- Flatpickr для вибору дати
